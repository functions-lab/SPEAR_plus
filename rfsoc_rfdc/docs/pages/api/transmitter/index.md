# Transmitter

DAC channel wrappers and transmitter task orchestration for single-channel,
multi-channel (synchronised), independent multi-channel, and MIMO
configurations.

| Module | Description |
|--------|-------------|
| [tx_channel](tx_channel.md) | Base Tx DMA channel wrapper |
| [tx_channel_real2real](tx_channel_real2real.md) | Real-to-Real passthrough |
| [tx_channel_iq2real](tx_channel_iq2real.md) | IQ-to-Real conversion |
| [tx_data_generator](tx_data_generator.md) | OFDM / detection packet generator |
| [single_ch_tx_task](single_ch_tx_task.md) | Single-channel transmitter task |
| [multi_ch_tx_task](multi_ch_tx_task.md) | Synchronised multi-channel Tx |
| [multi_ch_tx_indept_task](multi_ch_tx_indept_task.md) | Independent multi-channel Tx |
| [multi_ch_tx_mimo_task](multi_ch_tx_mimo_task.md) | MIMO transmitter task |
