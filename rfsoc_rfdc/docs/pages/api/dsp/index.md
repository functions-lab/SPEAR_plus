# DSP

Digital signal processing utilities for OFDM modulation/demodulation,
Zadoff-Chu preamble detection, CFO estimation, SNR calculation, and MIMO
detection.

| Module | Description |
|--------|-------------|
| [ofdm](ofdm.md) | OFDM signal generation and analysis (16/256 QAM) |
| [detection](detection.md) | Zadoff-Chu detection, CFO & SNR estimation |
| [mimo_detection](mimo_detection.md) | Multi-antenna MIMO detection |
