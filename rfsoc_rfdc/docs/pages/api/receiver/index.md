# Receiver

ADC channel wrappers and receiver task orchestration for single-channel,
multi-channel (synchronised), independent multi-channel, and MIMO
configurations.

| Module | Description |
|--------|-------------|
| [rx_channel](rx_channel.md) | Base Rx DMA channel wrapper |
| [rx_channel_real2iq](rx_channel_real2iq.md) | Real-to-IQ conversion |
| [rx_channel_real2real](rx_channel_real2real.md) | Real-to-Real passthrough |
| [rx_analyzer](rx_analyzer.md) | Rx data analysis base classes |
| [ofdm_analyer](ofdm_analyer.md) | OFDM / packet analysis |
| [single_ch_rx_task](single_ch_rx_task.md) | Single-channel receiver task |
| [multi_ch_rx_task](multi_ch_rx_task.md) | Synchronised multi-channel Rx |
| [multi_ch_rx_indept_task](multi_ch_rx_indept_task.md) | Independent multi-channel Rx |
| [multi_ch_rx_mimo_task](multi_ch_rx_mimo_task.md) | MIMO receiver task |
